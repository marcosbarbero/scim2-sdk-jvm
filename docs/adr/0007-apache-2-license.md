# ADR-0007: Apache 2.0 License

## Status

Accepted

## Context

We need an open-source license for publishing the SDK to Maven Central. The license must be business-friendly to encourage adoption in commercial projects.

## Decision

We will use the Apache License, Version 2.0.

## Consequences

### Positive

- Business-friendly and permissive — allows commercial use, modification, and distribution
- Patent grant protects both contributors and consumers
- Widely recognized and accepted in enterprise environments
- Compatible with most other open-source licenses

### Negative

- Requires preservation of copyright notices and license text in distributions
- Less copyleft protection compared to GPL-family licenses (derivative works can be closed-source)
